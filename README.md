# ChallengeU_App

A React Native Android app to encourage users take viral and charity challenges, interacting with the Facebook API.

The app Back end is in this repo: (https://github.com/RedEarthBlueSky/challenges-backend)

## Tech Stack
* React Native
* Redux
* Koa 2
* MongoDB
* Passport.JS
* Amazon S3
* Facebook SDK
